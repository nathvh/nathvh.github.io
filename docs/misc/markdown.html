    Les bases

    Ces éléments sont ceux originellement décrits par les créateurs du Markdown.
    Ils sont pris en charge par tous les moteurs de rendus.

    Nom & Rendu Markdown HTML

    Titre 1

    		 # Titre 1 <h1>Titre 1</h1>

    Titre 2

    		 ## Titre 2 <h2>Titre 2</h2>

    Titre 3

    		 ### Titre 3 <h3>Titre 3</h3>

    Titre 4

    		 #### Titre 4 <h4>Titre 4</h4>
    Titre 5
    		 ##### Titre 5 <h5>Titre 5</h5>
    Titre 6
    		 ###### Titre 6 <h6>Titre 6</h6>
    Gras **Gras** <strong>Gras</strong>
    Italique *Italique* <em>Italique</em>
    Citation
    		 > Citation <blockquote>Citation</blockquote>
    1. Liste
    2. ordonnée
      		 1. Liste
      2.ordonnée      		 <ol>
      <li>Liste</li>
      <li>ordonnée</li>
      <ol>
      • Liste
      • non ordonnée
        		 - Liste
        - non ordonnée        		 <ul>
        <li>Liste</li>
        <li>non ordonnée</li>
        <ul>
        Code `Code` <code>Code</code>
        		--- <hr>
        lien [lien](www.site.fr) <a href="www.site.fr">lien\

        Les compléments

        Ces éléments ont été rajoutés ensuite.
        Ils ne sont pas pris en charge par tous les moteurs de rendus et peuvent être plus ou bien implémentés.

        Nom & Rendu Markdown HTML
        Tableau C1Tableau C2
        		 | Tableau C1 | Tableau C2 | <table><tr>
        <td>Tableau C1</td>
        <td>Tableau C2</td>
        <tr></table>
        Entête tableau
        		 | Entête tableau |
        |---------------------|        		 <table><thead>
        <tr><th>Entête tableau</th><tr>
        </thead></table>
        Code
        multiligne        		 ```
        Code
        multiligne
        ```        		 <code>Code<br>multiligne</code>
        Note de fin de page [^1] & [^1]: /misc/descriptif pas d'équivalent

        Un super titre

        		 ### Un super titre {#perso} <h3 id="perso">Un super titre<h3>
        Mot
        Définition.
        		 Mot
        : Définition        		 <dl>
        <dt>Mot</dt>
        <dd>Définition</dd>
        </dl>
        Barré ~~Barré~~ <del>Barré<del>
        Case cochée - [X] Case cochée <input type="checkbox" checked> Case cochée</input>
        Case décochée - [ ] Case décochée <input type="checkbox"> Case décochée</input>
        😂 :joy: 😂
        Surligné ==Surligné== <mark>Surligné</mark>
        H2O H~2~O H<sub>2</sub>O
        X2 X^2^ X<sup>2</sup>

        Informations

        Pourquoi ?

        Le Markdown est un langage qui permet de mettre facilement en forme un texte et qui, contrairement à des langages plus lourds comme le HTML ou le XML, peut être assez facilement lu via son code.

        Il est très utilisé pour écrire de la doc, des release notes au readme.
